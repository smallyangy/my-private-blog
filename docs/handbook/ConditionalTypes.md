# 条件类型

## 类型

> ts中的类型 `code`

| 类型          | 语法形式                       |
| ------------- | ------------------------------ |
| 数字类型      | number                         |
| 布尔类型      | boolean                        |
| 字符串类型    | string                         |
| 符号类型      | symbol                         |
| Void类型      | void                           |
| Null类型      | null                           |
| Undefined类型 | undefined                      |
| Never类型     | never                          |
| 任意类型      | any                            |
| 数组类型      | T[]                            |
| 元组类型      | [T0, T1, ....]                 |
| 枚举类型      | enum T {...}                   |
| 函数类型      | (p1: T1, p2: T2) => T          |
| 类类型        | T                              |
| 构造器类型    | new (p1: T1, p2: T2, ...) => R |
| 对象类型      | {...} 或 interface T {...}     |
| 联合类型      | T1                             |
| 交叉类型      | T1 & T2 & ...                  |
